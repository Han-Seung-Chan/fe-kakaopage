#### 문제점

1. JS에서 알맞는 배열 인덱스를 데이터 파일에서 찾아서 넣어줘야함 -> 유지보수 불가능
2. 데이터 저장하는 파일에 키 값이 일정하지 않음
3. 함수를 반복적으로 실행 (반복문으로 처리 할 수 있지 않을까?)
